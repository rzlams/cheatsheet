SVG - Import and style as object 
                        .container svg path {
                          stroke-dasharray: 1000;
                          stroke-dashoffset: 1000;
                        }

                        .container.animate svg path {
                          animation: dash 5s linear forwards;
                        }

                        @keyframes dash {
                          to {
                            stroke-dashoffset: 0;
                          }
                        }
- Con este metodo se comienza a dibujar el trazo del svg hasta dibujarlo por completo

SVG - Dotted Round Border - El SVG escala segun su contenedor..
- Para modificar el tipo de borde se puede cambiar las propiedades stroke*.
- q representa una curva en el path.
- M representa MoveTo (en este caso el inicio) y L representa LineTo en el path
